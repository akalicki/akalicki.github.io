Stratus

Python Dropbox MongoDB

Free, unlimited cloud storage! Written in Python, using the headless WebKit PhantomJS to create new Dropbox accounts when the current one is full. The application leverages Dropbox's API to power data requests and a local MongoDB NoSQL database to store file information. The project works as a command line utility powered by familiar UNIX FTP-style controls while the background accounts created are abstracted away from the end user.

simple-gallery.js

JavaScript jQuery jQuery UI

A lightweight widget for quickly creating highly customizable photo galleries using standard jQuery notation. The idea behind the project was to separate the work of creating a cycling slideshow animation from the gallery's design process, allowing for greater facility of code-reuse and fewer headaches. simple-gallery.js creates a slideshow that you can easily customize without having to worry about superfluous code and unneeded options.

Tik Tok Tank

C# XNA Artificial Intelligence

A fun grid-based shooter I wrote in C# for the Game Design and Artificial Intelligence class I took my senior year in high school. It features both single-player and multiplayer modes, and tanks can be controlled using the keyboard or XBOX 360 controllers. Computer players feature very limited artificial intelligence and my own modified implementation of the A* search algorithm to help them navigate through the maze of blocks and find the human player using the shortest path possible.
