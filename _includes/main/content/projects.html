simple-gallery.js

JavaScript jQuery jQuery UI

A lightweight widget for quickly creating highly customizable photo galleries using standard jQuery notation. The idea behind the project was to separate the work of creating a cycling slideshow animation from the gallery's design process, allowing for greater facility of code-reuse and fewer headaches. simple-gallery.js creates a slideshow that you can easily customize without having to worry about superfluous code and unneeded options.

Calypso Farm Website

PHP MySQL HTML CSS

A larger-scale website and customized Content Management System designed from scratch using pure PHP and SQL for the Calypso Farm and Ecology Center located in Fairbanks, Alaska. Instead of starting with a commercial CMS and blog management system such as Wordpress that was bound to be bloated with more features than the nonprofit could hope to use, I set up a working SQL database on their server and designed a secure admin section that would better suit the organization's needs.
